## функциональность

- страница каталога содержит список велосипедов, доступных для аренды
- фильтр по пунктам проката скрывает велосипеды, не доступные по выбранному адресу, по-умолчанию не применен (Все пункты)
- в шапке каталога указывается количество найденных велосипедов (после фильтрации по пункту проката)
- если в списке больше 16 велосипедов, то отображаются только первые 16, а под каталогом появляется пагинация, позволяющая перейти на следующие страницы
- ссылка "На карте" открывает модальное окно с виджетом карты
- карточка велосипеда (клик по изображению или кнопке "Арендовать"):
  - если пользователь не авторизован, то открывает форму авторизации
  - если пользователь авторизован, то открывает модальное окно с детальной информацией о велосипеде
- кнопка "Арендовать" в модальном окне велосипеда:
  - если пользователь не ввел платежные реквизиты на этапе регистрации, то открывает форму заполнения реквизитов
  - если пользователь уже вводил платежные реквизиты, то показывает QR-код для оплаты

## дополнительная функциональность

- переключение формата отображения каталога (карточки / таблица)
- отображение доступных вариантов цвета в карточке велосипеда
- выбор цвета в модальном окне велосипеда
