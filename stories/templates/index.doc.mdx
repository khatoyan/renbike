## функциональность

- главная страница состоит из двух блоков (экранов):
  - приветственный экран "Сервис аренды велосипедов"
  - экран карты пунктов проката
  - высота экрана соответствует высоте вьюпорта браузера
- шапка неавторизованного пользователя:
  - логотип является ссылкой на главную страницу приложения (лендинг)
  - кнопка "Войти" показывает модальное окно с формой авторизации
  - кнопка "Регистрация" показывает модальное окно с формой регистрации
- приветственный экран:
  - кнопка "Выбрать велосипед" отправляет на страницу каталога
  - ссылка "Пункты проката" прокручивает страницу вниз до виджета карты пунктов проката
- экран карты
  - виджет карты содержит информацию о пунктах проката (на этапе верстки заменен картинкой-заглушкой)
  - кнопка "Выбрать велосипед" на виджете карты отправляет на страницу каталога с фильтром по указанному пункту выдачи
